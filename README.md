# ProjectLog
Capstone Project for intro to programming

This app stores peronal projects and gives them a status with tasks and descriptions for each project

Functions to build:
  Input function to create projects in database with descriptions 
  A function that classifies each function and what its for in the database
  A status function that the user can update in real time
  A task function that the user can edit and update in real time
  A function that sends each project to an archive database once it is finished
